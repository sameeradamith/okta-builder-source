Okta_builder_

Build your Okta startup templete with few steps!

Getting Started Guide

Using Okta_builder_ build your templete sucessfully. Download the file and unzip it.

1. Redirect Angular Applicaton

Run source application
  • Open Command Line or Terminal and navigate to project folder.
  • Prerequisites: Installed Nodejs and Angular in your local development environment.
  • Install dependencies using following command.
    • npm install
  • Then run application using following command.
    • npm start
    or ng serve --port:5000
Run Docker applcation
  • Open Command Line or Terminal and navigate to project folder.
  • Prerequisites: Installed Docker in your local development environment.
  • First, you need to build a Docker image.
    • docker build -t sample-angular-app-image .
  • Successfully build your image. Then run the application using the following command.
    • docker run -d -p 5000:5000 sample-angular-app-image

    You can customize the application name and port values according to your preferences.

Congratulations! You're almost done setting up your application. Visit this URL to test. (http://localhost:5000)
